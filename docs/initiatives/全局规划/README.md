# 长期规划入口

本目录用于维护“多项目控制面”这个长期主题的长期目标、总体路线和里程碑。

这里回答的问题是：

- 这个主题最终想达到什么结果
- 为什么现在做
- 分几个阶段推进
- 各阶段的大致目标是什么

这里不负责：

- 单次高风险变更的实施细节
- 某一阶段内部的详细门禁和冻结边界
- 架构、观测、Workflow 等专题 SPEC 正文

建议阅读顺序：

1. [10_愿景与成功标准.md](./10_愿景与成功标准.md)
2. [20_路线图与里程碑.md](./20_路线图与里程碑.md)

写作者路由：

- 写长期目标、成功定义、非目标：
  - 更新 `10_愿景与成功标准.md`
- 写高层路线图、阶段划分、长期里程碑：
  - 更新 `20_路线图与里程碑.md`
- 写某个阶段内部的详细门禁、冻结边界和交接要求：
  - 改到上一级的 `阶段规划/`
